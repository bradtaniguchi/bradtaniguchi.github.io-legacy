IMSRoomRenter Built with Python Built with Jetbrains

A python3 application that keeps track of student study rooms. This application is currently in use at California State University Dominguez Hills's Instructional Media Services Department. The application uses an sqlite3 database, and tkinter UI.

IMSTimesheet Built with Python Built with Jetbrains
A python3 application that keeps track of employee clock-ins and clock-outs. Easily customizable using a plain text file. Uses a sqlite3 database and provides a printable output file to review employee times. This application is currently in use at California State University Dominguez Hills's Instructional Media Services Department.

GoTorosBank Built with Java Built on Linux Built with Mysql Built with Bootstrap Built with Jquery

A Maven built java application that replicated a real life bank website. The application ran on a Linux server running Tomcat and Mysql. It also used Image recognition, and java's servlet technology. The front-end was done using Jquery and bootstrap. Created for California State University Dominguez Hills CSC 481 - Software Engineering.
